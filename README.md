# Mohamed Fadel — Portfolio

A cinematic Next.js portfolio built with TypeScript, Tailwind CSS, GSAP, ScrollTrigger, Lenis, and Canvas.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Personal media

- Add the portrait at `public/photo.jpg`. Until then, the photo reveal uses a designed `MF` fallback.
- Optional sounds can be placed in `public/audio/`. The intro is intentionally silent by default so browser autoplay policies never block entry.

## Editing

- English and Arabic content: `src/data/content.ts`
- Projects: `src/data/projects.ts`
- Colors and layout: `src/app/globals.css`
